# QuickDapp Docs

Documentation source for [quickdapp.xyz/docs](https://quickdapp.xyz/docs). Built with [Retype](https://retype.com).

## Development

```bash
npx retypeapp watch
```

## Build

```bash
npx retypeapp build
npx retypeapp serve
```

## Structure

```
getting-started.md          # Getting started guide
architecture-layout.md      # Architecture and project layout
environment-variables.md    # Environment variable reference
vibe-coding.md              # Vibe coding guide
backend/                    # Backend docs (bootstrap, auth, database, GraphQL, etc.)
frontend/                   # Frontend docs (components, theming, forms, etc.)
worker/                     # Background worker docs
users/                      # User management and notifications
variants/                   # Variant-specific docs (Web3, etc.)
deployment/                 # Deployment guides (Docker, binary, GitHub workflows)
command-line/               # CLI command reference
tutorials/                  # Step-by-step tutorials
```
